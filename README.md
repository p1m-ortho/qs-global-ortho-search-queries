> However, the methodology for rapid _scoping reviews_ remains undeveloped (PubMed `"rapid scoping"[ti]` search at the commit moment or on March 5, 2020, produced zero records; it is terrific to acknowledge that after 1.5 years, it produced **47 records** on October 6, 2021!
